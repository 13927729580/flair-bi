# Flair bi frontend

Following technologies are used:
* AngularJS 1.7
* Webpack
* npm
* Node 8+

### Development
Install dependencies 
```
npm i
```
In order to start development server you need to start following command:
```
npm start
```

### Adding new dependency
Running:
```
npm i <library> --save 
```
and importing in "src/vendor.js"

For dev tools
```
npm i <library> --save-dev
```
